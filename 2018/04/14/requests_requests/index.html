    requests主要接口

    数据获取 python requests 爬虫
    创建时间:2018-04-14 23:40
    1. 1.1 requests.request(method, url, **kwargs)
      1. 1.1.1选填参数:
      2. 1.1.2 **kwargs:控制访问的参数,共13个(选填参数)
    2. 2.1 其他6个接口
    3. 3.1 返回对象类型:

    Requests 所有的功能都可以通过以下 7 个方法访问。它们全部都会返回一个 Response 对象的实例。


    1.1 requests.request(method, url, **kwargs)

    • Constructs and sends a Request.

    1.1.1选填参数:

    method

    method for the new Request object.
    请求方式,对应http协议中的get , put , post等7种

    method = "GET"




    url

    URL for the new Request object.


    1.1.2 **kwargs:控制访问的参数,共13个(选填参数)

    params

    (optional) Dictionary or bytes to be sent in the query string for the Request.

    字典或字节序列,作为参数增加到url中

    >>>kv = {'key1':'value1','key2':'value2'}
>>>r = requests.request('GET','http://python123.io/ws',params=kv)
>>>print(r.url)




    data

    (optional) Dictionary or list of tuples [(key, value)] (will be form-encoded), bytes, or file-like object to send in the body of the Request.

    字典或字节序列或文件对象,向服务器提供或提交资源时使用

    >>>kv = {'key1':'value1','key2':'value2'}
>>>r = requests.request('POST','http://python123.io/ws',data=kv)
>>>body = '主体内容'
>>>r = requests.request('POST','http://python123.io/ws',data=body)



    json

    (optional) json data to send in the body of the Request.

    json格式的数据,作为request的内容

    >>>kv = {'key1':'value1'}
>>>r = requests.request('POST','http://python123.io/ws',json=kv)



    headers

    (optional) Dictionary of HTTP Headers to send with the Request.

    字典,http定制头字段

    >>>hd = {'user-agent':'Chrome/10'}
>>>r = requests.request('POST','http://python123.io/ws',headers=kv)




    cookies

    (optional) Dict or CookieJar object to send with the Request

    字典或CookiesJar,request中的cookies



    files

    (optional) Dictionary of ‘name’: file-like-objects (or {‘name’: file-tuple}) for multipart encoding upload. file-tuple can be a 2-tuple (‘filename’, fileobj), 3-tuple (‘filename’, fileobj, ‘content_type’) or a 4-tuple (‘filename’, fileobj, ‘content_type’, custom_headers), where ‘content-type’ is a string defining the content type of the given file and custom_headers a dict-like object containing additional headers to add for the file.

    字典类型,传输文件

    >>>fs = {'file':open('data.xls','rb')}
>>>r = requests.request('POST','http://python123.io/ws',files=fs)




    auth

    (optional) Auth tuple to enable Basic/Digest/Custom HTTP Auth.

    元组,支持http认证功能



    timeout

    (float or tuple)
    (optional) How many seconds to wait for the server to send data before giving up, as a float, or a (connect timeout, read timeout) tuple.

    设定超时时间,以s为单位

    >>>r = requests.request('GET','http://python123.io/ws',timeout=10)




    allow_redirects

    (bool)
    (optional) Boolean. Enable/disable GET/OPTIONS/POST/PUT/PATCH/DELETE/HEAD redirection. Defaults to True.

    True/False 默认为Ture,重定向开关

    • 默认情况下，除了 HEAD, Requests 会自动处理所有重定向。
    • 可以使用响应对象的 history 方法来追踪重定向。
    • 什么是重定向(如Github 将所有的 HTTP 请求重定向到 HTTPS)




    proxies

    (optional) Dictionary mapping protocol to the URL of the proxy.
    字典类型,设定访问代理服务器,可以增加登录认证

    >>>pxs ={'http':'heep://user:pass@10.10.10.1:1234'
          'https':'https:10.10.10.1:4321'}
>>>r = requests.request('GET','http://python123.io/ws',proxies = pxs)

    PS:你也可以通过环境变量 HTTP_PROXY 和 HTTPS_PROXY 来配置代理。

    $ export HTTP_PROXY="http://10.10.1.10:3128"
$ export HTTPS_PROXY="http://10.10.1.10:1080"

$ python
>>> import requests
>>> requests.get("http://example.org")




    verify

    (optional) Either a boolean, in which case it controls whether we verify the server’s TLS certificate, or a string, in which case it must be a path to a CA bundle to use. Defaults to True.
    verify:True/False,默认为True,认证SSL证书开关




    stream

    (optional) if False, the response content will be immediately downloaded.

    True/False,默认为True,获取内容立即下载开关



    cert

    (optional) if String, path to ssl client cert file (.pem). If Tuple, (‘cert’, ‘key’) pair.

    本地SSL证书路径


    2.1 其他6个接口

    requests.get(url,params=None,**kwags)

    发送一个GET请求

    requests.head(url)

    requests.head()获取HTML网页的头信息

    requests.post(url,data=None,json=None,**kwargs)

    向HTML页面提交post请求的方法

    requests.put(url,data=None,**kwargs)

    向HTML页面提交put请求的方法

    requests.patch(url,data=None,**kwargs)

    向HTML页面提交局部修改请求

    • patch适合局部更新(优点:节省带宽),put也可以不过要把全部字段一同提交,没有提交的字段被删除


    requests.delete(url,**kwags)

    向HTML页面提交删除请求

    3.1 返回对象类型:

    Response object
    具体Response object可查相关文档
